# Tkinter Calculator

This is a simple desktop calculator created using Python and the Tkinter library. The application provides a basic graphical interface to perform common arithmetic operations such as addition, subtraction, multiplication, and division.

The project was built to practice Python GUI development and understand event-driven programming using Tkinter.

## Technologies Used
- Python
- Tkinter (GUI library)

## Features
- User-friendly calculator interface
- Basic arithmetic operations
- Clear and backspace functionality
